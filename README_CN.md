# Blind Spatial Protocol (BSP)
## 盲操作空间协议：纯文本大语言模型无视觉、无坐标幻觉、零权限桌面控制

### 📄 论文预印本
- **论文标题**：Blind Spatial Protocol: Pure-Text LLM Desktop GUI Control via ElementMap and UIA Element ID
- **作者**：Yuanhui Gao (高沅会)(GitHub: @gaowatch)
- **版本**：v1.0 (Concept Paper)
- **GitHub仓库**：https://github.com/gaowatch/veyranova
- **论文PDF**：[bsp-v1.0-arXiv.pdf](https://github.com/gaowatch/veyranova/blob/main/bsp-v1.0-arXiv.pdf)

### ✨ 核心创新（根治坐标幻觉！）
1.  **ElementMap 盲操作协议**：通过结构化元素映射，让纯文本LLM直接引用UIA元素ID定位GUI
2.  **零坐标猜测**：从根本上消除视觉Agent的坐标幻觉问题，操作100%精准
3.  **零视觉、零权限**：无需截图、无需系统 elevated 权限，普通用户开箱即用
4.  **完整工具调用链路**：全流程解析LLM指令，确保操作精准落地
5.  **宪法级安全前置检查**：内置不可违反的安全规则，杜绝恶意操作，保障系统安全
6.  **纯文本LLM原生支持**：无需多模态模型，任何纯文本大模型均可直接使用

### 🎯 关联项目
本论文对应开源桌面Agent项目 **VeyraNova**，正在持续开发中，核心实现基于BSP协议。
